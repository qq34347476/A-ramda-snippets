## 使用方法

比如 R.adjust

按文档 使用 R.adjust 可在联想查看。

按返回值分类 R.arr 可查看所有返回值为 Array 相关函数

运算类，如 R.defaultTo，R.defaultTo 或 R| 都可触发联想 

## 分类片段

| 内容            | 快捷键 |
| :-------------- | -----: |
| 显示全部        |     R. |
| 返回值为Array   |  R.arr |
| 返回值为String  |  R.str |
| 返回值为Boolean | R.bool |
| 返回值为Number  |  R.num |
| 返回值为Object  |  R.obj |
| 查找类函数      |   R.fi |
| 排序(比较)函数  |   R.so |
| 流处理函数      | R.flow |
